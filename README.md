# 🐍 Python Creator Tribute Program

A simple Python program created to show respect and appreciation for **Guido van Rossum**, the creator of Python.

This project demonstrates basic **Object-Oriented Programming (OOP)** concepts while displaying information about Python and its creator.

## 📌 About The Project

The program creates a class that stores:

- Programming language name
- Creator details
- Inspiration behind the name

It then displays the information using an object method.

## ✨ Concepts Used

✅ Class and Object  
✅ Constructor (`__init__`)  
✅ Instance Variables  
✅ Instance Methods  
✅ Printing Output  

## 🛠️ Technology Used

- Python 3.x

````

## ▶️ How To Run

Run the program using:

```bash
python python_creator.py
````
